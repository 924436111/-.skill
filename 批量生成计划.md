## 每页执行规范

```
1. 读 PRD → 理解布局和状态
2. 读 pencil-prototype skill（DSL 语法）
3. batch_design 构建页面
4. snapshot_layout 校验
5. read ui-ux-pro-max skill → 检查对比度/触控/间距
6. get_screenshot 确认
7. 同步更新 Style Guide（如有新组件）
8. 同步更新对应 PRD（如有偏差）
```

## 核心约束

- 间距统一 16px（页面内卡片间距）
- 页面间间距 40px
- 所有文字色：`#F1F5F9` / `#94A3B8`
- 卡片：`fill:#1E293B, stroke:#334155, cornerRadius:16`
- 导航标题用 `layout:"none"` + 手动居中
- 发光统一 `#FFFFFF15`
- 清醒/醒来统一 `#F59E4B`（橙色警示）
