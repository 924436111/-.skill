## 首页仪表盘 · 功能描述

### 页面定位

用户打开 App 的第一屏，一站式概览昨晚睡眠全貌。信息密度适中，核心数据一眼看完。

### 状态判定（优先级从高到低）

App 启动时按以下顺序判定首页展示哪个状态：

```
flowchart TD
    A[App 启动] --> B{设备已配对?}
    B -->|否| C[空态·无设备]
    B -->|是| D{蓝牙已开启?}
    D -->|否| E[蓝牙未开启]
    D -->|是| F{本次连接成功?}
    F -->|同步中| G[同步中]
    F -->|传输失败| H[同步异常]
    F -->|成功| I{上次同步距今?}
    I -->|>7天| J{首次恢复连接?}
    J -->|是| K[弹窗:超过7天数据丢失]
    K -->|点知道了| L[数据过期]
    J -->|否| L
    I -->|1~7天| M[设备不在身边]
    I -->|<1天| N{有昨日数据?}
    N -->|否| O[空态·无数据]
    N -->|是| P{数据完整?}
    P -->|部分缺失| Q[部分数据缺失]
    P -->|完整| R{有心率/血氧异常?}
    R -->|是| S[心率/血氧异常]
    R -->|否| T[正常·有数据]
```

| 优先级 | 判定条件 | → 状态 |
| --- | --- | --- |
| 1 | 未配对 | 空态·无设备 |
| 2 | 蓝牙关闭 | 蓝牙未开启 |
| 3 | 正在拉取数据 | 同步中 |
| 4 | 连接成功但传输失败 | 同步异常 |
| 5 | 首次恢复连接 + 上次>7天 | 超过七天弹窗 → 数据过期 |
| 6 | 上次同步 1~7 天前 | 设备不在身边 |
| 7 | 已配对但无昨日数据 | 空态·无数据 |
| 8 | 部分指标缺失 | 部分数据缺失 |
| 9 | 心率\<50或>100 / 血氧\<90% | 心率/血氧异常 |
| 10 | 以上均不满足 | 正常·有数据 |

### 各区域功能说明

| 区域 | 内容 | 交互 |
| --- | --- | --- |
| **每日贴士** | 灯泡图标 + 睡眠健康建议文字，每日轮换 | 无交互，纯展示 |
| **睡眠评分卡片** | 5层叠加评分环：白光放射状发光 → 渐变圆角底色环(cyan→深色) → 中心遮罩圆 → 深色覆盖弧(控制进度) → 居中评分文字。大数字评分(cyan) + 评级(紫色)。设备胶囊：圆形图标 + 名称"智能手环" + 电量进度条(85%)。底部三栏统计：总时长 / 入睡·醒来 / 效率 | 点击进入睡眠详情页；点击设备胶囊 → 未配对→蓝牙搜索页，已配对→设备管理页 |
| **心率/血氧双指标** | 图标(心率红/血氧青) + 名称 + 大数值 + 单位/范围 | 点击进入对应指标的详情趋势页 |
| **睡眠阶段图** | 8时段垂直柱状图：清醒(#F59E4B, 橙色警示)、浅睡(#818CF8)、深睡(#C4B5FD)、REM(#22D3EE)，底部时间轴(23:00~06:00) + 颜色图例。右上角"详情 >"入口 | 点击进入睡眠详情页 |
| **底部导航** | 首页 / 趋势 / 改善 / 我的，当前页高亮(cyan)，其余灰色 | 切换页面 |

### 页面跳转与交互

首页作为中枢，所有可点击区域均有明确的跳转目标：

```
flowchart LR
    A[评分卡片] -->|点击评分环/三栏| F[睡眠详情·该日期]
    B[设备胶囊] -->|已配对| G[设备管理页]
    B -->|未配对| H[绑定设备页]
    C[心率指标卡] -->|点击| I[趋势·周报·心率tab]
    D[血氧指标卡] -->|点击| J[趋势·周报·血氧tab]
    E[睡眠阶段·详情>] -->|点击| F
```

| 点击区域 | 目标页面 | 携带参数 | 返回行为 |
| --- | --- | --- | --- |
| **评分环** | 睡眠详情·对应状态 | 当前日期 | ← 返回首页 |
| **评分三栏统计** | 睡眠详情·正常 | 当前日期 | ← 返回首页 |
| **设备胶囊**（已配对） | 设备管理 | — | ← 返回首页 |
| **设备胶囊**（未配对） | 设备管理·绑定设备 | — | ← 返回首页 |
| **心率指标卡** | 趋势·周报，锚定心率 tab | 周报+心率 | ← 返回首页 |
| **血氧指标卡** | 趋势·周报，锚定血氧 tab | 周报+血氧 | ← 返回首页 |
| **睡眠阶段·详情>** | 睡眠详情 | 当前日期 | ← 返回首页 |
| **每日贴士** | 无交互 | — | 纯展示 |
| **底栏·首页** | 当前页 | — | 高亮态，不跳转 |
| **底栏·趋势** | 趋势·周报 | — | tab 切换 |
| **底栏·改善** | 改善建议 | — | tab 切换 |
| **底栏·我的** | 个人中心 | — | tab 切换 |

> **不可点击区域**：每日贴士、标题文字、空态插图、状态栏。  
> **空态特殊处理**：无设备时评分区替换为"去配对"按钮 → 绑定设备页；无数据时三栏统计显示"--"仍可点击，目标页展示空态。

### 每日贴士轮换库

每日一条，按序或随机轮换。共 20 条，覆盖 5 个主题：

| # | 主题 | 贴士内容 |

| 4 | 环境 | 睡前 30 分钟调暗灯光，帮助褪黑素自然分泌 |

| 7 | 饮食 | 咖啡因半衰期约 5 小时，建议 14:00 后不再喝咖啡 |
| 8 | 饮食 | 少量温牛奶或香蕉含天然助眠成分，适合睡前 1 小时食用 |
| 9 | 运动 | 每天 30 分钟有氧运动可缩短入睡时间，但睡前 2 小时避免剧烈运动 |
| 10 | 运动 | 晚间拉伸或瑜伽有助于放松肌肉、降低心率，进入睡眠状态 |
| 11 | 设备 | 手环佩戴松紧以能插入一指为宜，太松影响心率读数准确性 |
| 12 | 设备 | 手环电量低于 20% 时及时充电，避免夜间记录中断 |
| 13 | 设备 | 睡眠音响 Pro 内置 12 种白噪音，可在 App 中切换 |
| 14 | 健康 | 成年人建议每晚 7-9 小时睡眠，长期少于 6 小时增加健康风险 |
| 15 | 健康 | 深睡占总量 20-25% 为理想范围，此阶段身体修复最活跃 |
| 16 | 健康 | 夜间醒来 1-2 次属正常，超过 3 次可能提示睡眠连续性差 |

## 页面状态

| 状态 | 触发条件 | 界面表现 |
| --- | --- | --- |
| **空态-无设备** | 新用户未配对 | 每日贴士常驻；评分卡片区域显示空态："绑定手环，开启睡眠追踪" + "连接智能手环后，自动同步睡眠数据" + 青色「去配对」按钮；心率血氧/睡眠阶段不显示 |
| **空态-无数据** | 已配对但设备无数据 | 每日贴士常驻；评分卡片：渐变环黯淡(0%覆盖)、评分显示"--" + "等待数据"、设备胶囊正常(已连接+85%电量)、蓝色提醒条"💡 佩戴手环入睡，明早自动同步数据"、三栏统计均显示"--"；心率/血氧显示"-- bpm" / "--%"占位；睡眠阶段显示"暂无阶段数据" + 灰色占位柱 |
| **正常-有数据** | 已配对且有数据 | 完整展示所有区域，数据为实际值 |
| **数据过期** | 超过 7 天未同步（本地数据显示但已陈旧） | 评分卡片正常展示旧数据，顶部显示"上次更新：7天前"，底部可能出现提示"数据可能不完整" |
| **蓝牙未开启** | 系统蓝牙关闭 | 顶部常驻黄色提示条，其余区域正常展示本地已有数据 |
| **心率/血氧异常** | 心率 \<50 或 >100 / 血氧 \<90% | 对应指标数字标红，加⚠图标 |
| **部分数据缺失** | 手环只采集了部分指标 | 缺失指标显示"-- 暂无数据"，其余指标正常 |
| **同步中** | App 后台拉取设备数据 | 顶部胶囊转圈 + "更新中…"，不影响其他区域正常展示上次数据 |
| **同步异常** | 连接成功但数据传输失败 | 顶部橙色提示"上次更新失败，将自动重试"，其余区域正常展示本地数据 |
| **设备不在身边** | 扫描不到设备 | 不提示错误；设备胶囊显示设备名+电量（若有缓存），顶部时间戳改为上次同步时间如"昨天 22:15"；其余正常展示本地数据 |
| **超过7天未连接** | 设备缓存溢出 | 首次连接成功后弹窗："手环仅保存最近7天数据，xx月xx日之前的数据已丢失" +「知道了」按钮，点击关闭弹窗进入首页 |

## 原型实现说明

*   原型文件：`pencil-new.pen`，三个页面：`zavvZ`(正常)、`P7hHK`(空态·无设备)、`vPyRM`(空态·无数据)
*   评分环技术：5层叠加（glow→gradRect→hole→cover→center），无法使用渐变描边，用 rectangle+cornerRadius 模拟渐变底色环
*   设备胶囊：w=182，圆角30px，内部圆形图标+名称+进度条
*   💡 emoji：Pencil 不支持 SVG 嵌入 icon\_font，暂用 emoji 替代图标（已知限制）
*   左右滑日期：规划中，原型未实现
*   数据过期/蓝牙未开启/异常指标/部分缺失：规划中，原型未实现
*   底部导航：4项(首页/趋势/改善/我的)，每项40×32 bg框+20×20 path图标+11px标签。活跃：#22D3EE20 bg + #22D3EE图标文字；非活跃：透明bg + #94A3B8。shadow blur:24 color:#00000040
*   状态栏：时间"9:41" + 电池图标(24×12)，所有页面统一
*   清醒/醒来色：#F59E4B（橙色警示），非灰色
