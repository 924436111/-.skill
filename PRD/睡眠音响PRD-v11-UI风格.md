# 睡眠音响 PRD v11 - UI 风格（Nocturne 紫）

> 版本：v11 | 日期：2026-06-05 | 阶段：E UI 风格确认（重新选型）

---

## 选定方案：D · Nocturne 紫

参照 Nocturne 设计系统，以薰衣草紫为主色调，搭配青色高亮与靛蓝辅色。

### 完整配色表

| 色值 | 名称 | 用途 |
|------|------|------|
| `#0F172A` | Neutral | 页面背景 |
| `#1E293B` | Surface | 卡片背景 |
| `#22D3EE` | Primary | 主色：评分数字、按钮、选中态 |
| `#C4B5FD` | Secondary | 强调：正向反馈、深睡 |
| `#818CF8` | Tertiary | 辅色：图表、清醒 |
| `#F1F5F9` | Text Primary | 标题、正文主文字 |
| `#94A3B8` | Text Secondary | 辅助说明文字 |
| `#64748B` | Text Muted | 标签、微文字 |
| `#334155` | Border | 卡片描边 |

### 各色深浅梯度

**Primary (`#22D3EE`)**
`#CFFAFE` → `#A5F3FC` → `#67E8F9` → `#22D3EE` → `#06B6D4`

**Secondary (`#C4B5FD`)**
`#F3E8FF` → `#DDD6FE` → `#C4B5FD` → `#A78BFA` → `#8B5CF6`

**Tertiary (`#818CF8`)**
`#E0E7FF` → `#C7D2FE` → `#A5B4FC` → `#818CF8` → `#6366F1`

**Neutral**
`#64748B` → `#475569` → `#334155` → `#1E293B` → `#0F172A`

### 睡眠分期配色

| 分期 | 色值 |
|------|------|
| 深睡 | `#C4B5FD` |
| 浅睡 | `#A5B4FC` |
| REM | `#22D3EE` |
| 清醒 | `#818CF8` |

## 字体

| 层级 | 字号 | 字重 | 行高 | 颜色 | 用途 |
|------|------|------|------|------|------|
| 大数字·评分 | 48px | 700 | 1.2 | `#C4B5FD` | 睡眠评分、核心KPI |
| 页面标题·卡片 | 20px | 600 | 1.3 | `#F1F5F9` | 页面标题、区域标题 |
| 指标·按钮 | 14px | 600 | 1.2 | `#22D3EE`/`#F1F5F9` | 心率/血氧数值、按钮 |
| 正文内容 | 16px | 400 | 1.5 | `#CBD5E1` | 卡片说明、列表项 |
| 辅助说明 | 13px | 400 | 1.4 | `#94A3B8` | 时间戳、范围值、图表标注 |
| 微标签 | 11px | 500 | 1.3 | `#64748B` | 图表轴标签、角标、底栏 |

**字体家族**：Plus Jakarta Sans（400 / 500 / 600 / 700）

## 组件规范

### 按钮

| 类型 | 填充 | 描边 | 文字 |
|------|------|------|------|
| Primary | `#C4B5FD` | 无 | `#0F172A` · 600 |
| Secondary | `#1E293B` | `#334155` 1px | `#F1F5F9` · 600 |
| Inverted | `#F1F5F9` | 无 | `#0F172A` · 600 |
| Outlined | 透明 | `#C4B5FD` 1.5px | `#C4B5FD` · 600 |

### 小按钮

| 类型 | 填充 | 描边 | 文字 |
|------|------|------|------|
| Primary | `#C4B5FD` | 无 | `#0F172A` · 600 |
| Secondary | `#1E293B` | `#334155` 1px | `#94A3B8` · 600 |
| Danger | `#EF444410` | `#EF444430` 1px | `#EF4444` · 600 |

### 搜索输入框

- 填充 `#1E293B`，描边 `#334155` 1px
- 圆角 24px（胶囊形），高度 48px
- 占位文字 `#64748B`

### 进度条

- 轨道：`#334155` 填充，高 8px，圆角 4px
- 填充色按分期颜色

### 底栏导航

- 背景 `#1E293B`，圆角 36px（胶囊形）  
- 激活态：图标 + 文字 `#C4B5FD`，背景微光 `#C4B5FD20`
- 非激活态：`#64748B`

## 图标集

**选定：Remix Icon** (`ri`)

| 属性 | 值 |
|------|-----|
| 风格 | 实心填充（纯面表达） |
| viewBox | `[0, 0, 24, 24]` |
| fill | 继承上下文颜色 |
| 节点类型 | Pencil `path`，`geometry` 提取自 SVG `<path d="...">` |

### 核心图标

| 用途 | 图标名 | 用途 | 图标名 |

| 趋势追踪 | `ri:bar-chart-fill` | 心率 | `ri:heart-pulse-fill` |
| 设置 | `ri:settings-fill` | 指南 | `ri:book-read-fill` |
| 日历 | `ri:calendar-fill` | 蓝牙 | `ri:bluetooth-fill` |
| 返回 | `ri:arrow-left-fill` | 通知 | `ri:notification-fill` |
| 同步 | `ri:loop-left-fill` | 设备 | `ri:device-fill` |
