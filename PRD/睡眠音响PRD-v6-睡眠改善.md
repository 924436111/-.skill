### Tab 1：改善建议

**生成规则**

| 触发条件 | 建议内容 | 优先级 |
|----------|----------|--------|
| 入睡时间偏差 > ±1h | 设定固定就寝时间，建议结合音响定时播放助眠音 | 高 |
| 深睡占比 < 15%（7天均值） | 睡前1小时远离屏幕，进行放松活动 | 高 |
| 睡眠时长 < 6h（7天均值） | 提前30分钟上床，保证至少7小时 | 中 |
| 醒来次数 ≥ 3次（7天均值） | 检查卧室环境（光线、噪音、温度） | 中 |
| 夜间心率均值 > 75 bpm | 避免睡前摄入咖啡因/酒精，尝试睡前深呼吸 | 低 |
| 血氧最低值 < 90% | 关注呼吸健康，建议咨询医生（不加入计划，仅提醒） | ⚠ 警告 |
| 连续 7 天评分提升 | 正向鼓励卡片："您的睡眠在改善，保持！" | 🎉 鼓励 |

> 每次最多展示 3 条。优先级：警告 > 高 > 中 > 低 > 鼓励，同类按数据异常程度排序。

**计划管理规则**

| 规则 | 说明 |
|------|------|
| **目标来源** | 从建议卡片"加入计划"导入，或手动添加自定义目标 |
| **目标上限** | 最多同时 3 个活跃目标 |
| **打卡方式** | 每天手动勾选当天格子 |
| **补打卡** | 允许补打昨天，不可补打更早日期 |
| **周期** | 按自然周（周一到周日），每周日晚上总结 |
| **完成判定** | 每周完成 ≥5/7 天视为达标 |
| **过期** | 连续 4 周完成率 < 30%，自动弹出是否移除该目标 |
| **提醒** | 每日 21:00 推送："今晚记得执行改善计划哦"（可关闭） |

## 页面状态

| 状态 | 触发条件 | 界面表现 |
|------|----------|----------|
| **建议-有数据** | 有 ≥3 晚数据 | 按规则生成 1~3 条建议卡片 |
| **建议-数据不足** | 不足 3 晚 | 提示："需要至少 3 晚睡眠数据来生成建议" |
| **建议-全部达标** | 所有指标正常 | 鼓励卡片："您的睡眠指标一切正常，保持下去！" |
| **建议-已有计划** | 某建议已加入计划 | 对应卡片按钮变为"已在计划中"，不可重复添加 |
| **计划-空态** | 未添加任何目标 | 空态插图 + "从改善建议中添加您的第一个目标" |
| **计划-有目标** | 有活跃目标 | 展示目标列表 + 打卡日历 + 统计 |
| **计划-目标已满** | 已有 3 个目标 | "添加新目标"按钮置灰："最多同时进行3个目标" |
| **计划-已过期提醒** | 连续4周<30% | 目标卡片底部出现橙色文字"完成率偏低，是否移除此目标？" +「移除」按钮 |
