## 组件规范

### 底栏导航
- 高度 72px，圆角 36px，padding [8,12]
- 活跃态：背景 `#22D3EE20`，图标+文字 `#22D3EE`
- 非活跃态：背景 `#1E293B`，图标+文字 `#94A3B8`
- 阴影：外阴影 offset(0,8) blur:24 color:#00000040
- 图标：20×20 SVG path，Remix Icon filled 风格

### 状态栏
- 高度 30px，时间 "9:41" 14px/600，电池图标 24×12
- 颜色 #F1F5F9

### 评分环（OnXba 组件）
- 5 层叠加：glow 径向渐变 → 渐变圆角底色环 → 中心遮罩圆 → 覆盖弧（控制进度） → 居中分数文字
- 大号：118×118（首页）、小号：100×100（详情）

### 卡片
- 背景 #1E293B，圆角 16，padding 20，gap 14
- 边框 #334155，strokeWidth:1，strokeAlignment:inner
- 强调卡片：border #22D3EE20

### 按钮
| 类型 | 背景 | 边框 | 文字 |
|------|------|------|------|
| 主要 | #22D3EE | — | #0F172A |
| 次要 | #22D3EE15 | #22D3EE | #22D3EE |
| 危险 | #EF444415 | #EF4444 | #EF4444 |
| 禁用 | #1E293B | #334155 | #64748B |

## 交互状态

| 状态 | 视觉表现 |
|------|----------|
| 正常 | 默认样式 |
| 按下 | opacity 0.8 / 高亮背景 |
| 禁用 | opacity 0.38 + 灰色 |
| 警告 | #F59E4B 边框/文字 |
| 危险 | #EF4444 边框/文字 |

---

## 深色模式规范

| 规则 | 标准 |
|------|------|
| 主文字对比度 | ≥ 4.5:1（#F1F5F9 on #1E293B = 12.6:1） |
| 辅助文字对比度 | ≥ 3:1（#94A3B8 on #1E293B = 4.7:1） |
| 弱化文字对比度 | ≥ 3:1（#64748B on #1E293B = 3.2:1） |
| 功能色不只靠颜色 | 警告/正常状态需配合图标或文字 |
