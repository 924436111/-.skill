# 睡眠音响 App 工作流

## 文件约定
- PRD → `PRD\`，命名：`睡眠音响PRD-v{版本号}-{模块名}.md`
- 设计导出 → `设计导出\`
- 原型文件 `.pen` → 项目根目录
- 先骨架后血肉，逐模块细化

## 工作阶段
1. 阶段 A（冷启动）→ 方向选择
2. 阶段 B（需求收敛）→ 逐条提问确认
3. 阶段 C（概要产出）→ 页面流转图 + 功能清单
4. 阶段 D（模块细化）→ 逐模块展开（功能→交互→状态）
5. 阶段 E（UI 风格确认）→ 配色对比 → Style Guide → 图标选定
6. 阶段 F（原型绘制）→ Pencil MCP 绘制 + 自校验
